cloud_request_to_gateway(msg_type, args, prefix='')[source]

Creates the payload for the cloud request to a gateway. This function is not meant for production, only for testing purposes.

Does

  1. Depending on the msg_type it will call a specific function or another, to create the payload.

  2. It will return this payload to the main create_message function.

Parameters

  • msg_type (int) – message type

  • args (list) – arguments that we wish to include in the message

  • prefix (string) – string to prepend to all the prints/console logs

Returns

payload

(bytes) – partial payload to be sent, without the message id and type, which are already sent
