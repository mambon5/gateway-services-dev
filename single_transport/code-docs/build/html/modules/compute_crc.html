compute_crc(payload, nbtyes=4)[source]

I got the CRC and checksum from https://reveng.sourceforge.io/crc-catalogue/all.htm. We choose the algorithm CRC-32/XFER because 1. The algorithm is evidenced by a widely-available implementation that can calculate CRCs for any desired message. as the webpage ensures us. 2. It returns exactly 4 bytes (which is the length we desire).

Parameters

  • payload (bytes) – message payload to convert to CRC

  • nbytes (int) – number of bytes to compress to in the CRC compression

Returns

CRC (string) – CRC-32/XFER CRC of 4 bytes

Formula

CRC-32/XFER https://reveng.sourceforge.io/crc-catalogue/all.htm
