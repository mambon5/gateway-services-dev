database_functions

Set of functions that control the interaction from main script to the database. We have the following functionalities implemented:

  1. Modify database values

  2. Read database values

sql_set_values(values, column_names)[source]

Creates part of the SQL SET statement. In particular, it creates the part of the query where each value is matched to its variable like so:

  • variable = value

Parameters

  • values (list) – List of values to be sent in the SQL statement

  • column_names (list) – List of database variables or column names to be updated

Returns

text (string) – the part of the SQL statement that does the variable assignation to the values
