update_gw_database(msg, mode=None, interlocutor=None, interac=None)[source]

Function that updates the gateway database.

does

  1. It checks the interlocutor (cloud/nodes)

  2. It calls either the:

    • update_gw_database_nodes() if the interlocutor is nodes

    • update_gw_database_cloud() if the interlocutor is cloud

Parameters

  • msg (RxMsg) – Message received class with all the arguments

  • mode (string) – either “listen” or “request”

  • interlocutor (string) – whether node is “cloud” or “nodes” mode

Returns

result (list) –

  • [0] - bool (True if successful, False if failed)

  • [1] - bool True if database is changed, False if no changes were made

  • [2] - list return arguments. Depending on the message type these arguments can be more or fewer
