read_gw_database(msg, mode=None, interlocutor=None)[source]

Function that reads the gateway database.

Does

  1. It checks the interlocutor (cloud/nodes)

  2. It calls either the:

    • read_gw_database_nodes() if the interlocutor is nodes

    • read_gw_database_cloud() if the interlocutor is cloud

Parameters

  • msg (RxMsg) – Message received class with all the arguments

  • mode (string) – either “listen” or “request”

Returns

result (list) –

  • [0] - bool (True if successful, False if failed)

  • [1] - list return arguments. Depending on the message type these arguments can be more or fewer
